                        PLANNING DOCUMENT
----------------------------------------------------------------------
1. Look at different company websites to get ideas
2. Create "draft" sketch of website layout in pencil + annotate
3. Create final sketch of website in color pens and markers + add annotations
4. Use your photo of your own for logo and don't forget to cite yourself
5. Create assignment folder: include sketches and README.txt & sub folders(css, html, images)
6. Create pages in html folder(index, about, lessons, contact)
7. Work on index html: 
    i. in head create META tags & descriptive title & add logo
    ii. in body create header (logo, nav with list where menu items are linked to their related pages), 
        main (articles-> headings, links that direct to headings->id, use of id and class selectors, paragraphs, images->figure, figcaption, alt)
    iii. in footer add copyright symbol and your name
8. Work on index css:
    i. use pseudo class selector 
9. Work on about html & css:
    i. same as step 7 & step 8
    ii. but use pseudo element selector to style 1st letter of paragraphs 
10. Work on lessons html & css:
    i. same as step 7 & step 8
    ii. but creat a striped table(td, tr, th + merging) using with contextual selector 
11. Work on contact html & css:
    i. same as step 7 & step 8
    ii. but create form with attribute selectors
12. Cite all sources
13. Validate all pages!
